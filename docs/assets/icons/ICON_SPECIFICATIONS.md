# Icon Specifications for AtlasMesh Presentation
## All Icons Required in White Color (PNG Format)

---

## Icon Requirements

**Format:** PNG  
**Color:** White (#FFFFFF or rgba(255, 255, 255, 1.0))  
**Background:** Transparent  
**Recommended Size:** 128x128px (scalable to 64x64px, 256x256px)  
**Style:** Modern, minimalist, line-based or filled icons  
**File Naming:** `icon_[name].png` (all lowercase, underscores)

## Icon Sources & Creation Methods

### Option 1: Free Icon Libraries (Recommended)
1. **Flaticon** (https://www.flaticon.com/)
   - Search for icons, filter by "white" or download and convert to white
   - Free with attribution, premium available

2. **Icons8** (https://icons8.com/)
   - Large collection, can customize colors
   - Free with attribution

3. **Font Awesome** (https://fontawesome.com/)
   - Convert SVG to PNG, set color to white
   - Free tier available

4. **Material Icons** (https://fonts.google.com/icons)
   - Google's icon set, can export as PNG
   - Free to use

5. **Heroicons** (https://heroicons.com/)
   - Modern, minimalist icons
   - Free, open source

### Option 2: Create Custom Icons
- **Tools:** Figma, Adobe Illustrator, Inkscape
- **Process:**
  1. Create icon in vector format (SVG)
  2. Set stroke/fill to white (#FFFFFF)
  3. Export as PNG with transparent background
  4. Ensure 128x128px base size

### Option 3: Convert Existing Icons
- **Tools:** ImageMagick, GIMP, Photoshop
- **Process:**
  1. Download icon in any color
  2. Use color replacement tool to make white
  3. Remove background (make transparent)
  4. Resize to 128x128px
  5. Export as PNG

## Quality Checklist

Before using icons, ensure:
- [ ] Icon is pure white (#FFFFFF)
- [ ] Background is transparent (no white box)
- [ ] Size is 128x128px (or scalable)
- [ ] File format is PNG
- [ ] Icon is clear and recognizable at small sizes
- [ ] Icon matches the modern, professional aesthetic
- [ ] File size is optimized (< 50KB per icon)

---

## Usage in Presentation

Icons should be placed:
- **Position:** Top of each panel, centered
- **Size:** Approximately 64-80px in the presentation
- **Spacing:** 20-30px margin below icon, before title
- **Alignment:** Centered horizontally within panel

---

## Notes

- All icons should maintain visual consistency (same style, line weight, etc.)
- If using icons from different sources, ensure they look cohesive
- Consider creating a custom icon set for brand consistency
- Test icons on both light and dark backgrounds to ensure visibility
