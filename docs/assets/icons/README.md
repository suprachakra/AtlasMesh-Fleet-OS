# Presentation Icons Directory

This directory contains all icon assets for the AtlasMesh presentation.

## Required Icons

All icons must be:
- **Format:** PNG
- **Color:** White (#FFFFFF)
- **Background:** Transparent
- **Size:** 128x128px (base size, scalable)

## Icon Files Needed

1. `icon_fleet_monitoring.png` - Wi-Fi/Signal icon
2. `icon_autonomous_dispatch.png` - AI/Person with gear icon
3. `icon_predictive_maintenance.png` - Wrench/Spanner icon
4. `icon_energy_management.png` - Battery/Lightning bolt icon
5. `icon_remote_operations.png` - Gamepad/Remote control icon
6. `icon_multi_fleet_coordination.png` - Circular arrows/Network icon

## How to Add Icons

1. Download or create icons matching the specifications in `ICON_SPECIFICATIONS.md`
2. Ensure icons are white color with transparent background
3. Save as PNG files in this directory
4. Use the exact file names listed above

See `ICON_SPECIFICATIONS.md` for detailed specifications, sources, and creation methods.
